Web服务SQL注入
==========================

.. _wssqli_concept:

简介
-----

Web服务通过SOAP请求进行通信。这些请求尝试执行用WSDL（web service description language）定义的函数。我们将要学习一些关于WSDL文件的知识。请参考WebGoat的WSDL文件进行学习。

.. _wssqli_goal:

课程目标
----------

查看WSDL文件并试着得到多个用户的信用卡号码。你不会看到返回的结果，一旦你确信你已经成功，刷新页面，你会发现成功的标志。
