XML注入攻击
============

.. _xmli_concept:

简介
-------

本课程将要指导你如何进行XML注入攻击。

.. _xmli_attack:

攻击原理
--------

AJAX应用程序使用XML来与服务器交换信息。这个XML很容易被恶意攻击者分析和篡改。
